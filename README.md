# hospital-emergency-room-information

## 응급실 실시간 정보
- 현재 위치에서 가장 가까운 응급실은 어디에 있는지를 알려줍니다. 더나아가 필요한 수술이 현재 가능한 상태인지 체크할 수 있습니다.

## 사용방법
-  프로젝트 빌드 후 google api key와 공공데이터 포털의 응급실 실시간 정보 API key을 발급받아 입력 후 실행
